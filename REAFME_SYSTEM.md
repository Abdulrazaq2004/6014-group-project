1. System Access

The Cyber6014 Game can be accessed in two ways:

Option 1: Online (Recommended)
The system is hosted online and can be accessed using the following link:
https://cyber6014.web.app/

No installation is required. The game runs directly in the browser.

Option 2: Local Installation

Download the project ZIP file.

Extract (unzip) the files to any folder on your device.

Open the file index.html using any modern web browser (Edge for better performance).

2. User Registration & Login

To access the system, users must create a new account.

Registration requirements:

Email address

Username

Password must:

Be at least 8 characters long

Contain at least one uppercase letter

Contain at least one lowercase letter

Contain at least one number

Once registration is complete, the user can log in using their credentials and access the full system.

3. System Features Overview

After logging in, users can:

Select cybersecurity categories, the only section is valid for testing is (phishing).

Choose difficulty levels (Easy, Medium, Hard).

Answer interactive quiz questions. 

Unlock higher difficulty levels by completing challenges.

Earn points and track progress.

View scores and compete with other users via the leaderboard.

The system is designed to be simple and intuitive, allowing users to explore features naturally without complex instructions.

The Cyber6014 Game is a web-based application and has been tested on:

Google Chrome

Microsoft Edge (best performance) 

Desktop and mobile devices
